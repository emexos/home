Moded - OS

What do you need, for a Mod-OS?

    1. if you want to make your own Mod-os simply download the OS and then install the following packages: 

      ' make, i686-elf-gcc, i686-elf-ld, nasm, and qemu '/ (homebrew) (mac/linux)
    2. open the Terminal in the OS directory and type in 'make', and if you don't want to install make then run ''./build.sh'.

    It doesnt work?

    - before ''./build.sh' run 'chmod +x build.sh', but make sure that you are in the right directory

if you don't want to use qemu,you can use VirtualBox.

OS - HelpMenu 
        clear     - Clear the screen
        about     - More about emexOS
        echo      - Show a message on terminal
        shut      - Shutdown the system
        reboot    - Reboot the system
        syscheck  - Show system specs
        gfx       - Starts emexOS GUI
        crfile    - Create a file
        rfile     - Read file content
        info      - Show a file info
        rename    - Rename a existing file
        delfile   - Delete a existing file
        ls        - List all files/dir in current local
        crdir     - Create a directory
        deldir    - Delete a existing directory
        cd        - Change the current directory
