ModOS

What do you need, for a Mod-OS?

    1. if you want to make your own Mod-os simply download the OS and then install the following packages: 

      ' make, i686-elf-gcc, i686-elf-ld, nasm, and qemu '/ (homebrew) (mac/linux)
    2. open the Terminal in the OS directory and type in 'make', and if you don't want to install make then run ''./build.sh'.

    It doesnt work?

    - before ''./build.sh' run 'chmod +x build.sh', but make sure that you are in the right directory

    now you can Modify the OS

if you don't want to use qemu, you can use VirtualBox.

About emexOS

  • Minimal Footprint; Pure C & Assembly, ~1 MB kernel image size
  • Memory Management; Simple malloc/free allocator in RAM
